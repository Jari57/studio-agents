## 🏗️ Quick Start

### Prerequisites
- Node.js 18+
- Firebase project
- Google Gemini API key
- Stripe account (for payments)

### Installation

```bash
# Clone repo
git clone https://github.com/your-org/studio-agents.git
cd studio-agents

# Backend setup
cd backend
npm install
cp .env.example .env
# Add your API keys to .env
npm start

# Frontend setup (new terminal)
cd ../frontend
npm install
npm run dev
```

Visit `http://localhost:5173`

### Environment Variables

**Backend** (`backend/.env`):
```env
PORT=3000
GEMINI_API_KEY=your_key_here
FIREBASE_PROJECT_ID=your_project
FIREBASE_PRIVATE_KEY="your_key"
STRIPE_SECRET_KEY=your_key_here
```

**Frontend** (configured in code - see `constants.js`)
