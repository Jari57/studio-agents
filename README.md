### Prerequisites
- Node.js 18+
- Firebase project
- Google Gemini API key
- Stripe account (for payments)

### Installation

```bash
# Clone repo
git clone https://github.com/your-org/studio-agents.git
cd studio-agents

# Backend setup
cd backend
npm install
cp .env.example .env
# Add your API keys to .env
npm start

# Frontend setup (new terminal)
cd ../frontend
npm install
npm run dev
```

Visit `http://localhost:5173`

**Backend** (`backend/.env`):
```env

FIREBASE_PRIVATE_KEY="your_key"
STRIPE_SECRET_KEY=your_key_here
```
